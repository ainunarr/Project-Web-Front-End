About Us

Saya memiliki keinginan untuk membuka usaha sendiri, seperti cita-cita saya yang ingin menjadi Wirausahawan. Salah satu usaha yang ingin saya tekuni ialah menjadi barista, membuka usaha cafe sendiri sepertinya menarik. Disini saya sangat ingin membuka cafe sendiri dan menjadi barista yang turun langsung dalam melayani para pelanggan di cafe.

Awal mula ketertarikan saya ini bermula ketika saya diberikan tugas untuk membuat project website, saya selalu memilih untuk membuat website dengan tema Coffee Shop, dan dari sinilah saya punya pemikiran ingin menjadi pengusaha yang memiliki tempat usaha sendiri.
